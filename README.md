# Practice: Mixed-Type Arrays

## Description
This project demonstrates how to store different data types (strings, numbers, booleans, and arrays) within a single JavaScript array. 

## Key Learnings
* **Mixed Types:** Storing varied data in one structure.
* **Nested Arrays:** Accessing an array located inside another array using double brackets: `array[index][subIndex]`.
* **Array Modification:** Updating values by index and using `.push()` to expand nested lists.
